3.2媒体查询（Media Query）

***新语法，针对不同媒体类型定义不同的样式，针对不同的屏幕尺寸设置不同的样式

xxxxxxxxxx

    

    

    

    

        

            

            

            
@media mediatype and|not|only (media feature){

        

        
CSS-Code;

        
}
mediatype的取值：

1 .all ——用于所有设备

  1. print ——用于打印机
  1. screen ——用于电脑屏幕，平板电脑，智能手机
关键字：

***and——将多个媒体符连接到一起，即“且”，常用

not——排除某一种媒体类型，可以省略

only——指定某一种媒体类型，可以省略

media featrue的取值：

width——可视宽度

min-width——最小宽度

max-width——最大宽度

xxxxxxxxxx

    body{

        

            
        background-color:black;

        

        
    }

        
}

        

            
/*当我们的电脑屏幕小于800px时，网页颜色为黑色*/
